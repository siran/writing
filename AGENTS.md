# Agent Instructions

## Shell commands
- Do not use compound commands (e.g. `cmd1 && cmd2`, `cmd1 ; cmd2`).
  Run each command separately. Compound commands trigger extra security
  validations in this environment.

## Language / framing
- Never describe the torus as a "support," "toroidal support," or container.
  The torus is the emergent shape of self-refracting flow closing on itself.
  Use "toroidal closure" or "self-refracting closure." See Ch 8.
- The book derives known physics from first principles. Use "recovered" not
  "identified" when describing results derived from source-free transport.
